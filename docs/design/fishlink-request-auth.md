ログイン／新規登録／送金先登録（農家の出品準備まで）のレビューが終わりました。**既存実装の修正＋一部新規**です。

仕様の正本：
- 仕様書：**`fishlink-auth-spec.md`**
- モック：**`fishlink-auth-mockup.html`**（①〜⑦認証／⑧〜⑩送金先・計10画面）

## 全体像
- **電話番号＋パスワード**でログイン（ログインID＝ユーザー名は廃止。識別子は電話番号）。
- **セッション維持**（一度ログインしたら入りっぱなし）。
- **パスワード回復＝SMS-OTP**（電話→コード→新パスワード）。SMSが使えない場合のみ運営手動。
- 新規登録は**即時利用OK**（審査なし・不正は運営が事後対応）。本番デフォルト言語＝**クメール語**。

## 3. パスワード回復（新規フロー・SMS-OTP）
1. 電話番号を入力 →「コードを送信」
2. SMSで届いた**6桁コード**を入力 → 検証
3. **新しいパスワードを設定** → そのままログイン
- 安全策：コード有効期限（例10分）＋試行回数制限＋**未登録番号でも「送りました」と曖昧表示**（番号の有無を漏らさない）。
- ※SMSが使えない場合の暫定＝運営が番号確認して手動リセット。

## 4. 登録後オンボーディング（再構成）
登録の直後に1ステップで案内（各ステップにアイコン）：
- **① ホーム画面に追加**：iPhone（Safari）／Android（Chrome）で手順が違うため**端末別の図解**（本番は端末を自動判別して該当手順だけ表示）。
- **② 通知を許可**：押すとOS許可ダイアログ。**拒否済みなら「設定から許可」導線に切替**。※iOSは①の後でないと通知が効かない（①→②の順）。
- **③ 端末の音をオン**：操作の図（マナーモード解除＋音量UP）のみ。
- **農家のみ：送金先を登録**（出品前に必須＝下記5）。「あとで設定」可。

## 共通
- **言語**：本番デフォルト＝**クメール語**。**埋め込みGoogle地図の言語もアプリ言語に同期**（`language=km/en`・現状クメールでも地図UIが日本語）。
- **色**：CTA＝青／緑はGAqP・成功専用／赤はdanger専用／amberは注意・要件。
- **バリデーション**：インライン方式（未入力欄を赤＋メッセージ＋最初のエラーへスクロール・ボタン無効化しない）。※送金先の出品ゲートだけ例外で投稿ボタン無効。

## 確認したいこと
- **🔴 SMS（OTP送信）がカンボジアで使えるか・コスト**。用途は**パスワード回復だけ**（ログインは電話＋パスワードでSMS不要）＝送信量少・低コスト。マスク通話（Twilio等）と同じ通信インフラなので、まとめて確認したいです。
- **Firebaseの内部ID（メール）を"ユーザー名ベース"から"電話番号ベース"に変えられるか**（電話番号から合成メールを生成・標準的だが実装変更）。
