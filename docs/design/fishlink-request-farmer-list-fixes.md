## 確認①：配達範囲外の農家が一覧に出ている

実装で「テスト農家モバイル（Yamaguchi・3679.5km）」「大星農場（Tokyo・4457.8km）」が一覧（おすすめ上位含む）に出ていました。

仕様（§6）では **自分に配達できる農家だけ表示**＝`distanceKm ≤ その農家の配達可能距離（deliverableRadiusKm）` の農家のみ。範囲外（買えない遠方農家）は**非表示**です（魚一覧で配達範囲外の商品を非表示にするベースルール list-spec §7 と同じ思想）。

→ 確認をお願いします：
- **(a)** この配達範囲フィルタが農家一覧に**適用されているか**（適用済みなら、上記は配達可能距離を極端に大きく設定したテストデータの可能性＝本番では消える）。
- **(b)** 未適用なら、**§6 のとおり `distanceKm ≤ deliverableRadiusKm` で絞り込み**を入れてください（範囲外農家は一覧に出さない）。

※Yamaguchi/Tokyo・debugトークン等はテストデータなので、本番データでの挙動で判断でOKです。

## 追加②：読み込みを「もっと見る」方式に（初期8件・+8件ずつ）

農家一覧の読み込みを **「もっと見る」方式**（魚一覧と統一）にしてください。

| 項目 | 内容 |
|---|---|
| 初期表示 | **8件**（調整可パラメータ 6〜10。1列の縦長カードなので魚一覧の10件より控えめ） |
| もっと見る | ボタンで **+8件ずつ** 追加表示 |
| リセット | **ソート／魚種フィルタを変更したら展開数をリセット**（先頭8件に戻す） |
| 保持 | 生産者ページから**戻ったときは展開数を保持** |
| 件数表示 | 「◯件の農家」は**該当の総数**（もっと見る前でも総数を出す） |

- **理由**：低速回線対策が主（農家カードはカバー写真＋アバターを持つ縦長カード＝全件表示だと農家が増えたとき写真を一度に大量読み込み＝重い）。＋農家増加への備え＋魚一覧と作法統一。
- モック反映済み（`fishlink-farmer-list-mockup.html`に「もっと見る」ボタン＋挙動。※モックは農家5件なので例示用に初期4件で表示。本番は8件）。

## 追加③：魚種チップに「その他」追加＋固定12種をそろえる

魚種チップ列を **固定12種＋「その他」** にそろえてください。

| 箇所 | 現状 | 修正後 |
|---|---|---|
| 魚種チップ列 | 11種・「その他」なし（「スポットパンガシウス」が抜け） | **固定12種＋「その他」**＝ライギョ／ヒレナマズ／レッドティラピア／ナイルティラピア／シルバーバーブ／**スポットパンガシウス**／パンガシウス／ジャイアントスネークヘッド／キノボリウオ／カエル／バラマンディ／オニテナガエビ ＋ **その他** |

- **「その他」を入れる理由**＝出品フォームは「その他」カテゴリ（CAAに無い魚）を投稿できる＝その他出品は実在する。チップが無いと、その他カテゴリしか扱わない農家がフィルタで見つからず埋もれる。買い手の魚一覧フィルタにも「その他」がある（list-spec §4）＝統一。
- 「その他」の判定＝`fishKinds` に「その他」を含む農家。選択中チップ青・該当タグ青強調は他の魚種と同じ。
- ついでに **固定12種が全部入っているか確認**（「スポットパンガシウス」が抜けていないか）。
- モック反映済み（`fishlink-farmer-list-mockup.html`：チップ13個＝12固定種＋その他。その他選択でロカ養殖がヒット）。
