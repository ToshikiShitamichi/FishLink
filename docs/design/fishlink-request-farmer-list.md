# 【実装依頼】農家一覧（おすすめ農家「もっと見る」の着地）新設＋遷移修正（2026-06-16）

Home「☆ おすすめ農家」の「**もっと見る ›**」を確認したところ、**農家を見たいのに魚一覧（商品カードのリスト）に着地**していました（農家→商品のミスマッチ）。本来の着地＝**農家カードを並べた「農家一覧」**を新設し、遷移を直す依頼です。

## 全体像

- **入口**＝Home「☆ おすすめ農家」の「もっと見る ›」。
- **着地**＝この**農家一覧**（農家カードを評価順に縦1列で並べた発見リスト）。
- **位置づけ**＝**子画面**（タブではない）。ヘッダーに戻る矢印＋「⭐ おすすめ農家」、ボトムナビは**ホームがアクティブ**。
- 農家ブラウズは設計上は副次的（リピート単位＝魚種）なので**軽い画面**にとどめる。絞り込みは**魚種だけ**。

> なぜ別画面か：「Homeの魚種タイル → 魚一覧（魚種で絞った商品）」は今日その魚を**買う**動線でOK（別依頼で対応済み）。一方この「おすすめ農家 → もっと見る」は良い農家を**発見する**動線で、農家カードの一覧が正解。両者は別物・別目的です。

## ② 農家一覧の新設

- **農家カード＝Home「おすすめ農家」のカードをそのまま流用**（カバー写真＋✓GAqP＋アバター＋農家名＋「州・地区 ・ km」＋★%(件)／新規＋取扱魚種タグ＋「農家を見る →」青枠）。**縦1列（全幅）**で並べる（2列にしない）。
- **並び（ソート）＝2択**：`評価が高い順`（デフォルト）／`距離が近い順`。価格順などは付けない。
- **魚種フィルタ＝上部に魚種チップ横スクロール行**（**文字のみ**・アイコンなし）。タップでその魚を**扱う農家**に絞り込み・再タップで解除・選択中は青＋該当タグ強調。これ1つだけ（地域・価格は付けない）。
  - 判定＝農家の取扱魚種（`fishKinds`＝現在＋過去の出品。今日売切れでもレパートリーに含む）。
- **新規農家（レビュー0件）**＝「新規」表示・評価順では末尾だが**隠さない**（埋もれ防止）。
- **配達範囲外の農家は非表示**（`distanceKm ≤ deliverableRadiusKm` の農家だけ＝自分に届く農家のみ。魚一覧の商品ベースルールと同じ）。
- **0件の空状態**＝「**◯◯を扱う農家はいません**」＋青「すべての農家を見る」（魚種フィルタ解除）。
- **タップ先**＝カード／「農家を見る →」＝**生産者ページ**（既存）。

## 共通・補足

- **色**＝青＝操作/選択/CTA（ソート選択・魚種チップ選択・該当タグ・「農家を見る」ボタン）。緑＝GAqP（認証）専用。評価★＝amber。
- 魚種チップを文字だけにした理由＝魚種の識別はHome「魚から探す」タイル（アイコン＋名前）が担う。絞り込みチップは全画面 文字で統一（魚一覧の絞り込みも文字）。実機で読みにくければ後でDS Species Icon Setを追加。
- テストデータ（test農家／Yamaguchi／3679.5km／宣材画像）は本番差し替え前提・仕様判断不要。

## 範囲外

- 「もっと見る」のページング（件数が増えた時）。MVPは一覧表示or「もっと見る」方式は実装判断。
- 重い絞り込み（地域・価格）は付けない（農家ブラウズは副次的）。需要が出たら後追い。
