# 【実装依頼】公開質問（Q&A）農家ビューの権限ゲート（2026-06-19）

下道さん

農家「投稿一覧」のカード →「💬 質問」→ 質問ページ（農家ビュー）を確認したところ、**買い手（レストラン）向けの質問入力欄がそのまま出ていました**。Q&A の権限は「**レストラン＝質問する／農家＝返信のみ**」なので、農家ビューでは質問コンポーザーを出さないのが正です（qa-spec §4）。

仕様の正本：
- 仕様書：**`fishlink-qa-spec.md`**（§2 フッターの出し分け／§4 コンポーザーの出し分け／§9 差分の「2026-06-19」表）
- モック：**`fishlink-qa-mockup.html`**（**⑤ 農家ビュー**面を追加＝コンポーザーなし・農家向け空状態）

## 修正点（農家ビューのみ・買い手ビューは現状維持）

| # | 箇所 | 現状（農家ビュー） | 修正後 |
|---|------|------|--------|
| ⑥ | 下部の質問コンポーザー | 「質問を入力／質問する／公開質問として商品ページに表示されます」が**表示**されている | **農家には非表示**（農家は新規質問しない＝返信専用）。質問が来たら各スレッドに「返信する」（農家は全スレッド返信可） |
| ⑦ | 空状態（質問0件） | 「まだ質問はありません／**最初の質問をどうぞ**」（買い手への促し） | 「まだ質問はありません／**レストランからの質問がここに表示されます**」（農家向けの待ち受け文言・入力欄なし） |
| ⑧ | 公開注意 | 「質問は他のレストランにも公開されます」 | 農家向けに **「レストランからの質問・あなたの返信は他のレストランにも公開され、注文前の参考として表示されます」**（誰の何が公開されるか農家視点で） |

## 補足

- 出し分けの判定＝**閲覧者がこの出品のオーナー農家かどうか**。オーナー農家＝返信専用（コンポーザー非表示）／レストラン＝質問可（現状維持）。
- 1アカウント1ロール（認証の決定）なので「農家アカウントが質問する」状況はありません。
- 買い手ビュー（質問コンポーザーあり・「最初の質問をどうぞ」）はそのままでOKです。
