## 補足
- **バナーのデータ源**：運営画面（料金設定）の `serviceRatePct` / `campaignEnabled` / `campaignDiscountPct` / `campaignEndDate` を参照。表示条件＝バナー表示ON ＆ campaignEnabled ＆ 今日≤終了日。**%・終了日はハードコードせず運営値を出す**（A〜Cはこの徹底で解決）。
- **色の役割（再掲）**：青＝価格/操作/リンク/CTA ／ 緑＝GAqP・成功・送料無料 ／ 赤＝danger（バナー外。Homeでは「支払期限/在庫切れ辞退」等の注記のみ） ／ amber＝割引・キャンペーン明細。
