## 共通仕様

**コメントのマスク＋モデレーション**（公開テキストのため）
- **連絡先マスク（自動）**：本文の電話番号・SNS・URL → 伏字（Q&A spec §6・チャットと共通ロジック）。両方向に適用。
- **モデレーション**：管理画面「**トラブル報告**」（Q&Aと共通）に、マスク発動コメントが「要確認」として上がる → 運営が **🗑非表示にする**（公開から消すがDBは残す）。新画面は作らない。
- ※連絡先を含まない誹謗中傷・虚偽の対策（通報・その場非表示等）は後フェーズ。

**評価の表示先**
- レストラン→農家＝**生産者ページ**に公開。
- 農家→レストラン＝**承認時に農家が見る「レストラン信頼スコア（★%）」**（承認待ち詳細に既にある評価%がこれ）。タップで開くレストランページは**別途・新規（後日）**。

**匿名性**
- MVPは**即時公開・非匿名**＋運営が初期は目視監視。報復が問題化したら**ダブルブラインド**（両者提出まで非公開）へ昇格（後送り）。
