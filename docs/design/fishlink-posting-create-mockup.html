FISHLINK 農家側 — 魚を投稿する（モック）

配送は農家ごとの設定（A案）：配送3点は魚ごとに変わらない＝農家の属性。一度設定すれば全投稿に自動適用、変更も1箇所で全投稿に反映。

① 初回（配送設定なし）＝フォームに配送3欄が出る → 投稿時に保存。　② 2回目以降（配送設定あり）＝配送欄は消えてサマリー1行＋変更に。

初期値：空＋必須=魚単価/配送可能距離/送料単価、プリフィル=内臓処理500、空＝任意=送料無料距離/最低注文量。価格・CTAは青に統一。
