## 全体像（ルール）
- 紹介コードは全員が持つ（固定）。**招待された新規が初回取引を完了すると双方に特典**。
- 役割別：レストラン＝**割引クーポン**／農家＝**受取ボーナス**。**双方付与**。
- 付与タイミング（Uber式）：招待された新規の**クーポンは登録時**（初回注文で使える）／**現金（農家ボーナス）と招待した人の特典は、招待された人の初回取引完了後**。

## 2. シェア
- シェア文は**紹介コードを埋め込んだ自動入力リンク**（例 `/register?ref=PwhUSjvC`）。受け取った人は**タップ→登録画面にコードが入った状態**（手入力不要）。
- 特典の文言は**今は入れない**（配布が動いたら具体額を入れて拡散力UP）。
- **シェア文はユーザーのアプリ言語（クメール語）で生成**。

## 3. カート（クーポン適用）
- **コード入力欄は撤廃** → **お持ちのクーポンをタップで適用**（マイページでコピーして貼る、をやめる）。
- 明細に「**🎟 紹介クーポン -2,500**」行。合計から2,500引く（**原資はプラットフォーム負担・農家は満額**）。最低注文額あり（運営設定）。
- **案Z（排他）**：**クーポン使用中はキャンペーン割引を出さない**（お得な方が自動適用＝通常クーポン2,500＞キャンペーン938）。クーポンを**外すとキャンペーン割引が戻り、明細・合計も変わる**。

- 整合ルール：紹介コード固定／1被紹介者に紹介元1人／既存ユーザーは後から被紹介されない（コード入力は新規登録時のみ）。
