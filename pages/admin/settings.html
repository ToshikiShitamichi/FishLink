メンテナンス
州データの正規化
既存ユーザーの province を内部キー（例: takeo）に統一します。実行は1回でOK。実行後は地域フィルターの重複が解消されます。
district のクメール語版を Geocoder で取得
既存ユーザーの位置情報から逆ジオコーディングして districtKm を埋めます。Khmer 表示時の市区町村が英語のまま残っている既存ユーザー向け。Geocoder API を呼ぶため、対象ユーザー数 × 約100ms かかります。
取引数（取引N件）を集計
既存の完了済み注文から各ユーザーの tradeCount（プロフィールの「取引N件」）を集計します。冪等・再実行可（タイムアウト時は再クリックで続きから）。以降の完了は自動で加算されます。
